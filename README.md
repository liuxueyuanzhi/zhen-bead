# 珍豆你玩

一个简洁的拼豆（Perler Beads/Hama Beads）像素画设计工具，支持智能生成拼豆示意图。

## 主要特性

### 绘图功能
- **自定义画布大小** - 支持 4x4 到 200x200 的任意尺寸
- **多种绘图工具** - 画笔、橡皮擦、填充工具、吸色器、直线、矩形、圆形
- **多种像素样式** - 圆形、方形、圆角三种样式切换
- **专业调色板** - 支持 HSL 颜色选择器、HEX 直接输入
- **本地图片转换** - 上传图片自动转换为像素画
- **拼豆数量统计** - 自动统计每种颜色所需数量
- **导出图片** - 支持导出为 PNG 图片

### 多色板与颜色管理
- **多色板预设** - 支持全色板、168色、144色、96色、48色快速切换
- **相似颜色合并** - 基于 HSL 色彩空间的智能颜色合并算法
- **多品牌色号系统** - 支持 MARD、COCO、漫漫、盼盼、咪小窝五大品牌
- **色号自动映射** - 根据选择的品牌自动显示对应色号

### 移动端优化
- **虚拟摇杆控制** - 左摇杆控制画布移动，右摇杆控制缩放
- **触摸优化** - 双指缩放、单指拖动画布
- **响应式布局** - 完美适配手机、平板等各种屏幕尺寸

### 选区操作
- **复制/粘贴** - 复制选区内容到剪贴板
- **剪切** - 剪切选区内容
- **清除** - 快速清除选区内容
- **反选** - 将选区反色

### 用户系统
- **登录认证** - 用户必须登录才能使用编辑器
- **使用期限管理** - 管理员可设置用户使用期限
- **AI 次数限制** - 管理员可设置用户 AI 生成次数上限

### AI 智能生成
- 输入描述或上传参考图，AI 自动生成像素画
- 密钥仅部署环境变量配置，客户端不包含任何第三方密钥

## 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite 6
- **后端**: Netlify Functions
- **数据库**: MongoDB Atlas
- **样式**: Tailwind CSS（CDN 加载）
- **AI**: 豆包 Seedream 模型

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到 Netlify

1. 将代码推送到 GitHub 仓库
2. 在 [Netlify](https://netlify.com) 连接仓库
3. 配置环境变量
4. 部署

### 环境变量

| 变量名 | 说明 |
|--------|------|
| `MONGODB_URI` | MongoDB Atlas 连接字符串 |
| `ADMIN_USERNAME` | 管理员用户名 |
| `ADMIN_PASSWORD` | 管理员密码 |
| `JWT_SECRET` | JWT 密钥 |
| `PIXELBEAD_AI_API_KEY` | AI API 密钥 |
| `PIXELBEAD_AI_API_BASE` | AI API 地址（可选） |
| `PIXELBEAD_AI_IMAGE_MODEL` | AI 模型名称（可选） |

## 管理员功能

管理员登录后可以：
- 管理用户账号
- 设置用户使用期限
- 设置用户 AI 生成次数
- 重置用户密码
