## 描述（App Store Description）

珍豆你玩是一款专为拼豆（Perler Beads / Hama Beads）爱好者打造的像素画设计工具。从灵感到图纸，一站搞定。

【核心功能】
- 自由画布：支持 4x4 到 200x200 任意尺寸
- 多种画笔：铅笔、橡皮擦、油漆桶、吸色器、直线、矩形、圆形
- 三种像素样式：圆形、方形、圆角，随时切换预览
- 专业调色板：HSL 色轮 + HEX 输入，覆盖各大品牌拼豆色号
- 拼豆数量统计：自动计算每种颜色所需数量

【智能生成拼豆图】
- 输入文字描述或上传参考图，一键生成拼豆风格示意图（应用内建服务，免费使用）
- 生成结果可导入画布后继续手工精修

【图片转拼豆】
- 上传照片自动转换为拼豆像素画
- 高级裁切：自由选区或 1:1 裁切
- 智能颜色合并：自动匹配最接近的拼豆色号

【多品牌色号】
- 支持 MARD、COCO、漫漫、盼盼、咪小窝五大品牌
- 全色板 / 168色 / 144色 / 96色 / 48色预设快速切换
- 可调相似颜色合并阈值

【沉浸拼豆模式】
- 全屏查看图纸，专注拼豆创作
- 网格线 + 辅助线 + 色号标注
- 颜色高亮定位，方便逐色拼装
- 锁定模式防止误操作

【分享与素材广场】
- 一键生成分享链接，好友打开即可查看
- 素材广场：浏览和收藏其他创作者的作品
- 导出高清 PNG 图纸，支持参考线和镜像翻转

开始你的拼豆创作之旅吧！

| 诊断 | 否 | |

### 第三方服务

- **Upstash Redis**: 存储用户主动分享的图纸数据（7天自动过期）
- **MongoDB Atlas**: 素材广场数据存储
- **Vercel Analytics**: 匿名访问统计
- **智能生成服务**: 用户主动使用「智能生成」时，描述或参考图会发送至开发者部署的服务端处理并返回图像（不在应用内配置第三方密钥）
